好玩：尝试转移到Ubuntu，失败

本文最后更新于：2020-01-07

今天尝试把主力机器换成Ubuntu的系统，但是失败了

想换Ubuntu的原因，一方面是想折腾玩玩，另一方面是前一段 CG Geek 的 Windows Vs Linux - Every Blender user needs to see this! 视频，说到Blender在Linux上比Win10快50%

折腾了一整天，由于没有技术，踩了无数坑

其中最奇葩的坑是 “Ubuntu找不到SanDisk U盘里系统文件”
做好启动U盘后，从U盘启动，不管是改bios设置、还是挨个试USB2.0/3.0接口，都提示错误

unable to find a medium containing a live file system

在网上找了各种方法，最后发现是SanDisk U盘的问题…..
解决方法在这里 When I start the Ubuntu ISO, initramfs says “unable to find a medium containing a live file system”

只有闪迪32G的U盘有这个问题，好沙雕的bug

历经千辛万苦，装好Ubuntu，又开始装AMD显卡驱动、调高分屏缩放问题、装小工具
问题越来越多….

我太弱了，还是乖乖回去用Windows吧
