笔记：三星 Galaxy SCR01 随身 Wi-Fi

本文最后更新于：2023-12-14

最近拿到一台三星 Galaxy SCR01 随身 Wi-Fi，折腾一下

知道这个设备，是因为之前在网上偶尔看过国内推荐这个东西的帖子
似乎因为小巧便携，续航优秀，而且价格不贵（400~500 人民币），在国内热度不低的样子

这个东西在日本好像是办随身 Wi-Fi 套餐送的，套餐解约之后就不能上网了
不过其实它只是在软件中做了封锁，万能的网友找到了办法，只要通过设备内的隐藏菜单解除封锁，随身 Wi-Fi 就又能愉快上网冲浪了

解除 SIM 锁，开启 5G

网上教程一搜一大把，随便找了一篇
知乎 - 光栅镜像 - 三星5g 随身Wifi scr01 安装第三方桌面、设置apn、解锁5g sa 图文教程

偷一段教程：

点击关于-状态-手机号码，快速点击10次
点击10次后，会进入DRParser Mode，此时依次输入 *#0011#
进入到service mode，再次点击右上角，点击一次back，再次点击key input
这时输入 Q 并确认，在此界面等待大概一分钟后，会进入到主菜单界面
点击[2] UE SETTING & INFO，点击[1] SETTING，点击[1] PROTOCOL，点击[9] NR，点击 [2] allow list control，点击[2] allow list off，显示APLY success，点击右上角菜单，点击两次back回到上层目录；点击 [1]nr5g sa/nsa mode control，点击[1] sa/nsa enable，按照提示重启设备

  • allow list off 应该就是有关 SIM 卡封锁的功能，默认情况下只能使用办套餐时给的 SIM 卡，把这个列表关掉之后，就可以使用任意运营商的卡

  • sa/nsa enable 用来开启 5G SA，好像速度比普通 5G 快？

添加 APN

完成上面的步骤之后，还不能上网
需要添加运营商对应的 APN，每个运营商的设置都不同，需要在网上搜 运营商名字 APN 设置，一般都能找到
然后去 通信设置 >> APN 里面按照网上的说明添加设置，右上角状态栏就应该出现 4G 或 5G 的字样，旁边有上下小箭头
这样就说明设置成功，可以网上冲浪了

有一个需要注意的地方
通信模式，在设置的时候最好选择默认的 Standard（ST）标准模式
Plus Area（+A）模式需要看运营商和所在的地区是否支持
手上的一张 SIM 卡在设置 APN 的时候，怎么都连不上网，后来发现换成 Standard（ST）模式就 ok 了，另一张卡实测 Standard（ST）和 Plus Area（+A）均可使用

一些非必要设置

如果只是当随身 Wi-Fi，到这里就完全可以了
国内的很多教程附带了 安装第三方桌面 和 设置手势 的操作

  • 安装第三方桌面
    安装其他桌面之后，这个设备就可以像一般的安卓手机一样，能看到系统设置，能自由安装应用

  • 设置手势
    设置手势，是因为如果不小心点到导航栏的方形按钮，设备会回到随身 Wi-Fi 默认的界面，在这个界面任何按键都没办法回到第三方桌面，除非重启设备
    这个时候就可以在界面上设置一个悬浮的小程序，操作这个悬浮窗回到自定义的桌面

所以，第三方桌面可以随便找，搜 Android launcher app，挑一个喜欢的，adb install ***.apk 安装上去
然后 adb shell am start -a android.intent.action.MAIN 唤起更改默认桌面的菜单，选自己安装的启动器

设置手势也可以找类似的，只要能显示在应用程序上面就可以
看的几篇教程里推荐的应用是 流体手势，我自己安装了一个 悬浮球，也能实现同样的效果
