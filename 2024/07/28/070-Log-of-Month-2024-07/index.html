《Python 工匠》

很棒的一本书，感觉又又又又又一次点燃了我学习 Python 的热情
搓出来一篇博客 学习：Python 学习资源推荐 & 学习笔记，以后准备把 Python 相关的内容都放在这篇文章下
文章里我对本书的介绍是这样的：

Python 非常自由，但相应的代价就是上下限差距会很大，这本书讨论如何把 能跑的代码 改成 优秀的代码、如何利用 Python 语言特性写出更加 Pythonic(Python风格) 的代码
还介绍了一些不止 Python，任何语言都通用的知识，例如变量命名、大型项目里如何统一代码风格
内容通俗易懂，GitHub 版本可以免费阅读，如果感觉讲的不错可以购买书籍版，比 web 版更加翔实

《Eloquent JavaScript》

Eloquent JavaScript 4th Edition by Marijn Haverbeke
还没看完，目前看到三分之一左右
简单介绍一下这本书的优点

  • 完全免费
    本书有 web 版、PDF 版、适用于手机的小 PDF 版、EPUB 版、MOBI 版，全部都可以在网站上免费获取
    当然也有纸质版，可以买一本放在书柜里
    所有版本的内容完全没有区别，甚至 web 版体验要更胜于其他版本，因为有代码高亮，以及下面另一个特点

  • 代码块可运行
    上面这么多版本中，第一次学习时最推荐使用 web 版
    因为 web 版的每一段代码块都是可以运行的，作者讲解知识点，给出一段代码示例，这时，你就可以点击这个代码片段，实际运行看看有什么效果
    而且还可以自由修改这些代码，试验能产生什么不同的效果
    边学边练，不用安装任何环境，甚至不用离开浏览器，最小化学习成本

  • 内容循序渐进，学习和实践结合
    从最基础的语法讲起，第一部分全面详细的介绍了 JavaScript 语言本体，第二部分是浏览器，第三部分 Node.js
    第一部分里还有例如 encapsulation(封装)、 polymorphism(多态)、 inheritance(继承) 三板斧，object-oriented(面向对象)、 functional(函数式编程)、 closure(闭包)、 recursion(递归)、 side effect(副作用) 等常见编程概念
    每章学习完毕都有三四个小练习题，直接在浏览器里做题
    隔几章会有个 Project 章节，作者手把手教你做个完整的小工程

    说实话，第一个 Project: A Robot 里运行 runRobotAnimation 时，看到屏幕上跑来跑去的小机器人有点感动

  • 网站简洁美观
    内容就是这本书，没有任何无关资讯，结构简洁明了，排版美观

这种时候真的想感谢互联网、感谢作者的互联网精神，消除信息传播壁垒，让每个人都能平等地获取优秀知识

这部电影放在当下看感觉很有意思，稍微记录一下

  • AI
    2009 年想象 AI 可以在网络世界兴风作浪，现在有 AI 水军/谣棍兴风作浪（
    AI 潮流确实影响了普通人的生活（Sell me this pen. It’s AI powered.
    当然，好处也有，这个月学 Python 和 JavaScript，遇事不决问 AI，体验非常愉快
    现在看来，AI 确实影响了所有人的生活，好坏两方面都有，至于哪方面影响更大，还真不好说（

  • 网络/现实生活分界消失
    影片里任何人可以在任何地方连接到虚拟世界，只要有 电脑、翻盖手机（或者 电视（
    这部电影上映于 2009 年，那个年代，我家里没有宽带，打游戏只能靠偷摸去网吧，手机行业里诺基亚还是绝对的霸主，wap 网页版 QQ农场还在流行
    在学校，电脑老师带来一台笔记本电脑加 USB 流量卡，实现了真正的 移动网络
    电脑老师还跟我们说有个叫 “淘宝” 的东西，可以网上购物
    当然，大家都觉得是骗人的，网上买东西？这能靠谱吗？
    上面这一切距离今天也就 15 年

    竟然已经到能说出这种话的年纪了，明明学生时代 15 年 就约等于整个人生

    似乎有点跑题（
    想说的是，智能手机确实延伸了网络的边界，网络生活侵入现实生活（或者反过来，现实生活融入网络生活？
    移动互联网让 上网 不再需要付出成本，大家都是 24 小时在线
    在这个时代，下网 反而变成非正常的状态，需要额外付出努力

    之前看到一个概念 互联网移民，指千禧年前出生的人，经历了网络普及的过程
    与之相对的是 互联网原住民，千禧年之后出生的人，对他们来说网络就和阳光、空气一样，是理所应当存在的事物
    挺有趣的概念

  • 虚拟身份反客为主
    虚拟身份某种程度上变得比现实身份更重要，这也是上面网络生活入侵的结果
    账号被盗的男主不能证明自己是自己，唯一的解决办法是找系统投诉，这很赛博朋克

  • 单点故障
    影片里 AI 盗取公共机构账号就能破坏公共设施，盗取军方人员的号就能操作导弹，实在是有点扯
    但结合这两周 CrowdStrike 全球蓝屏事件，似乎也没那么离谱（
    同样是中心化系统的脆弱性，看电影时忍不住把两者联想起来

  • 可视化
    影片开头一段虚拟世界的导览，现实生活中有实体的事物，例如书本、汽车，在虚拟世界里也有同样的东西

照骗

看 败犬女主太多了 第一话最后天台的片段，感觉非常有夏日氛围
于是问了一下 AI：如何调出有日系夏日氛围的照片？
AI 回答如下：

  1. 基础调整

    • 提高曝光度，增加整体亮度
    • 降低阴影，提亮暗部细节
    • 适度提高对比度，增加画面层次感

  2. 色温调整

    • 稍微偏冷，强调夏日清凉感
    • 色温向蓝色调整，但不要过度，保持自然

  3. 色彩处理

    • 提高蓝色和青色的饱和度，强化天空的蓝色
    • 轻微降低整体饱和度，呈现柔和的日系风格
    • 使用HSL面板，微调各个色彩的色相、饱和度和明度

  4. 曲线调整

    • 在RGB曲线中创建轻微的S型曲线，增加对比度
    • 在蓝色通道曲线中稍微提高亮部，强化天空的蓝色

  5. 分离色调

    • 高光部分添加淡蓝色调
    • 阴影部分可以添加轻微的暖色调，平衡整体色彩

  6. 细节调整

    • 适度增加清晰度，突出建筑和云朵的纹理
    • 控制降噪，保持画面干净

  7. 局部调整

    • 使用径向滤镜，轻微提亮建筑物，增加层次感
    • 使用渐变滤镜，稍微强化天空的蓝色和云朵的白色

  8. 添加颗粒感

    • 适度添加颗粒，营造胶片质感，增加日系氛围

  9. 最后调整

    • 微调整体色调平衡，确保画面和谐
    • 适当提高鲜艳度，增加夏日活力感
