Phoenix Documents

这个月开始看基于 Elixir 的前端框架 Phoenix
Phoenix 默认用 Postgres，要先安装、配置数据库，但我却被这个看起来简单（实际也非常简单的）步骤卡了一小时…

参考 Arch Wiki - PostgreSQL

看起来成功了，只需要 pg_ctl -D /var/lib/postgres/data -l logfile start 启动服务
但执行这个命令会报权限错误

看文档，查论坛，问 AI，试了各种方案都不行
直到最后，发现 Arch Wiki 下面还有一行小字 Finally, start and enable the postgresql.service.
被自己蠢到…
原来是用 systemctl 控制的…

《Eloquent JavaScript》

感觉 Eloquent JavaScript 里 异步 章节讲解不是很清晰，看了好几遍都没怎么理解 Promise、async/await，准备配合 MDN 文档作为补充资料

Others

最近看的电视剧/电影，点评尽量不涉及剧透

劇場版　少女☆歌劇 レヴュースタァライト

新文芸坐隔几个月就会重映 少歌剧场版，来日本后第一次看少歌就是在这里
时隔一年左右，再次来看重映，影片内容当然没什么好说的，一如既往眩しい✨

这次想说的是新文艺座的 WSBライティング上映
简单说，这个就是在放映厅挂满 RGB 灯条（光污染机箱），配合荧幕画面实时控制灯光效果，烘托气氛
放映前主持人惯例暖场时提到这个灯光是手动控制的，灯光师在会后面的放映室和大家一起看电影，同时控制灯光，因此每场放映会的灯光效果都不会完全一样
主持人是以非常自豪的语气说这些话的，但我听到之后第一时间想到的是「手动控制灯光？这怎么保证质量？手动控制的话不会延迟很高吗？」
疑问一旦生根就不会消失，整场电影里都在下意识地注意这个灯光，「嗯，确实有点延迟」、「这里效果有点一言难尽啊…」、「这个地方效果还可以」诸如此类的想法不断涌现，分散我的注意力
例如，皆殺しのRevue 地铁场景效果很好，路灯接连不断，突破荧幕闪过整个放映厅，模拟车窗外灯光飞掠的效果，有一点整个影厅都进入舞台中的感觉，沉浸感++
但还有些场景探照灯甩来甩去，晃得睁不开眼（物理意义上眩しい）

最后看下来，感觉灯光弊大于利，太分散注意力了，我更希望可以正常欣赏影片
顺便附上这篇帖子 『劇場版 少女☆歌劇レヴュースタァライト』 初見でWSBライティング上映を浴びた感想 - ①ライトアップ上映の感想，文章中对影院坚持手动控制灯光效果的解释我很喜欢，是我没有想到的角度

今思えば、態々こういった手間のかかる演出をして下さったのは「全く同じ舞台は二度と訪れない」という、スタァライトの秘めるメッセージ性を体現するためだったのかな……
大致翻译：现在回想起来，特地使用这种费时费力的表现方式，也是为了呼应少女歌剧「完全相同的舞台不会再度出现」这一暗示吧

胡思乱想

想换工作

理想中的派遣：体验各种工作，积累不同行业、编程语言经验
实际上的派遣：外行指导内行，最下游工作，不需要（并且日本人也不想让你）动脑子，纯搬砖

现在的工作再做十年也只是单纯重复而已，根本不能提高技术

最起码编程水平是很难提高的，能进步的只有 Excel 技术，毕竟日本人均 Excel 仙人，每天 面向 Excel 编程（

太没意思了，想找一份能磨炼技术的工作

游记

散步

又是拿来凑数的多摩川散步时间（

似乎是梅花？

两张图放一起，枝头赏月

一般见到的 止まれ 都是代表危险的红色牌子，偶然看到一个锈迹斑斑的灰色牌子，尤其是在色彩丰富的背景中突然出现一个完全没有色彩的物体，格格不入到好像是 PS 出来的，感觉很有趣

左上漂亮的蓝粉色云朵，右下斑驳水泥建筑
清水混凝土外墙本来应该是冷冷清清、高级感十足的，但黑色污渍加上去就变成一块脏脏的破抹布了

杂

花粉症、襲来

又到了花粉肆虐的季节
眼睛肿、流鼻涕、嗓子痛，一套组合拳打下来，每天起床都是噩梦的开始
吃药、戴口罩效果都聊胜于无，鼻涕从起床开始流到工位，通勤途中几分钟不擦鼻涕就要流嘴里了

有时候真的担心鼻涕流太多会不会脱水

垃圾工作 + 鼻涕连天，工作日精神状态堪忧😇

结语

拖更拖出新高度，三月底才写完二月份的月记
想一想，其实并没有特别难的部分，刚开始只是修图和少歌的评论需要花点时间，但一旦开始迟延 –> 二月的记忆逐渐模糊 –> 由于回忆模糊而难以下笔 –> 继续拖延，变成了这样的恶性循环，越拖越难，越难越拖…
比如动画部分的评论，二月份还在追的动画，有几部现在已感觉索然无味，没看下去了，但因为是二月份月记，还需要忠实地记录当时的感受，结果就是在三月底回想已经远去的二月份记忆，并且还要小心地剔除三月份记忆的影响

太难了，下次不能这么拖了（flag
