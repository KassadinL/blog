常用软件

本文最后更新于：2021-10-07

重装系统了，记录一下常用软件 (In Linux, Web browser and Android)

软件(In Linux)

  • v2ray

    上网必备
    切换到 su 环境，bash <(curl -L -s https://install.direct/go.sh)

    2021/02/24 update
    出来一个xray，似乎比v2ray功能更强大
    link: https://github.com/XTLS/Xray-core

  • Qv2ray

    v2ray GUI
    最近发现的v2ray前端，体验还不错

    Qv2ray在ubuntu gnome下可以自动设置系统代理，但是sudo命令的代理似乎还要配置一下

    笔记：Linux系统代理设置 最后更新了ubuntu的设置方法

  • Wacom

    数位板
    不算软件吧，linux默认识别wacom板子，只是有些设置还是需要调整，比如模拟鼠标滚轮、右键按下且点击后生效
    在 039 Log of migrating to Linux 里说过如何设置了

  • Telegram

    即时通讯
    2020/07/28 桌面版刚更新的2.2版本支持多账户，不需要多实例了，之前写的 笔记：Linux下运行多个Telegram实例

  • Firefox

    浏览器
    linux自带，倒是不需要手动装

  • mpv

    视频播放
    apt安装就可以，配置文件可以参考 038 MPV configuration 里列出的参考资料(没有写完，还没完全搞懂，咕咕咕….

  • okular

    看文件
    拿来看pdf、cbr都不错，能满足我的需求

  • aria2

    下载
    Github - P3TERX/ Aria2 一键安装管理脚本 增强版 大佬的一键脚本非常好用
    也可以自己apt安装 aria2，然后用大佬的 Github - P3TERX/ Aria2 完美配置

    2020/08/04 更新
    [P3TERX - Telegram Bot 控制 Aria2 下载并自动上传到 Google Drive、OneDrive]
    又出了一个aria2的telegram bot前端，p3大佬写了个教程，很详细
    我试着把这个写成systemctl daemon，但是跑不起来，报错/usr/bin/env: ‘node’: No such file or directory
    前两天查了查，没解决问题，今天又想到，换关键词试了试，找systemctl daemon的环境变量，还真是这个问题 How to set environment variable in systemd service?
    加一行 Environment="PATH=......"，把bash里的$PATH值加进去就好了

  • Rclone

    网盘同步
    可以apt安装
    如果是VPS，可以参考 P3TERX - Aria2 + Rclone 实现 OneDrive、Google Drive 等网盘离线下载，很好玩
    可惜墙国的大局域网，我安装rclone也就用来时不时下google drive文件

  • Putty

    ssh客户端
    apt安装，Manjaro上的配置文件copy过来竟然不能用，不太清楚哪里出问题了，只好重新配置一下，之前的帖子 笔记：Putty设置及使用公钥验证登录

  • youtube-dl

    视频下载
    下油管视频

  • you-get

    视频下载
    主要用来下b站视频，结合 -c cookies.sqlite 参数，可以读取firefox cookie文件，下载大会员清晰度视频

  • qBittorrent

    BT/PT
    迁移qBittorrent数据：Where does qBittorrent save its settings?
    但是把数据拷过去，qBittorrent开始自动校验资源
    太费时间（也费硬盘了），想跳过这步，在网上找了找，没找到特别简便的办法
    只能用苯方法，大概思路：找到qb准备校验的种子的torrent文件，删掉qb里的任务，用torrent文件重新添加任务，添加时选择skip hash check（也可以用magnet link，只是这个需要看网络状况，torrent文件就不用

    • ~/.local/share/data/qBittorrent/BT_backup/这个目录里保存qb里任务的种子，但是文件名是种子的hash

    • 在qb里选中需要的任务，右键copy hash，打开bash复制这些种子到一个方便的目录，删掉qb里的任务，重新添加
      （太难了，下载的资源分散在三个目录，BT/PT混杂，tag分了一堆，不然任务量会小很多

      2021/01/03 update
      Install qbittorrent-nox on Ubuntu server: How to Install qBittorrent on Ubuntu 18.04 Desktop or Server

      2021/10/07 update
      Github Wiki - Running qBittorrent without X server (WebUI only, systemd service set up, Ubuntu 15.04 or newer)

      Github Wiki - Installing qBittorrent
      qbittorrent-nox-static 4.1.9 version
      Linux服务器安装qBittorrent 4.1.9 nox (Ubuntu)

  • Rime

    ibus-rime中文输入
    不过ubuntu似乎自带，不用手动安装了，之前的帖子 笔记：配置ibus-rime中文输入法

  • Joplin

    笔记软件
    随便找了一个支持markdown预览的笔记软件

  • Hexo

    博客

浏览器插件

再补充一些正在用的浏览器插件

  • HTTPS Everywhere

    屏蔽http链接

  • Vimium

    键盘控制浏览器，减少鼠标/数位板操作

  • Red Block

    Twitter拉黑插件，功能丰富，可以blcok跟随/点赞，而且不会像block chain一样触发Twitter限制

  • SingleFile

    把整个网页保存为单个html文件

  • Tampermonkey

    油猴，安装解除b站区域限制脚本，可以看港澳台地区番剧，虽然现在看的少了

  • To Google Translate

    翻译

  • Google search link fix

    获取谷歌搜索结果的真实链接，而不是从谷歌跳转过去

  • uBlock Origin

    去广告插件，能去油管广告，似乎还有高级玩法，没有研究

  • User-Agent Switcher and Manager

    firefox不能看b站直播，V2EX - Ubuntu 下 bilibili 无法收看直播，帮帮孩子！ 帖子里说是ua的问题，这个插件换一下ua就可以了，有效

安卓(Android)

2021/08/18 update, some Android applications.

  • Bitwarden

    开源，密码管理软件，可以自建，也可以用官方的服务，7$/year
    我从1password迁移过来的，Bitwarden除了界面丑点，其他还可以，功能并没有太大差别

  • SagerNet / AnXary

    开源，翻墙软件，替代V2rayNG(听说这个不遵守开源协议)

  • F-Droid

    开源，软件商店，都是开源软件

    2021/08/19 update
    发现一个F-Droid替代软件Aurora Droid，界面美观，功能丰富
    最重要的是，可以配合Magisk加载Aurora Services服务，实现app后台更新，不需要在F-Droid里一个个点install了
    Aurora Services： https://github.com/whyorean/AuroraServices

  • KDE Connect

    开源，Linux KDE桌面和手机互通，手机之间也可以传文件之类的，还算好用

  • ClipboardCleaner

    开源，Android 9上可以开启一个服务，几秒钟之后自动清空剪贴板，但是 Android 11上这个功能不能用了

  • Fritter/NewPipe

    开源，分别替代 Twiiter/Youtube

  • KOReader

    开源，看书软件，专为墨水屏/电纸书/E-Ink设备优化

  • Scrambled Exif

    开源，分享图片时，移除图片Exif信息，并且重命名图片

  • VLC

    开源，播放器，Android版做的还挺好的

  • Magisk

    开源，面具，Root手机（这是个大坑，我还没开始跳

  • FairEmail / XPrivacy

    开源，欧洲软件，分别是邮件客服端、权限管理软件
    FairEmail支持PGP
    XPrivacy可以给软件假权限
    这两个是从网上看的，还没有试过，有时间研究一下
